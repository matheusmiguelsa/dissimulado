132. A company has an encrypted Amazon S3 bucket. An Application Developer has an IAM policy that
allows access to the $3 bucket, but the Application Developer is unable to access objects within the
bucket.

What is a possible cause of the issue?

A. The S3 ACL for the $3 bucket fails to explicitly grant access to the Application Developer

B. The AWS KMS key for the $3 bucket fails to list the Application Developer as an administrator
C. The $3 bucket policy fails to explicitly grant access to the Application Developer

D. The $3 bucket policy explicitly denies access to the Application Developer

Answer: D
